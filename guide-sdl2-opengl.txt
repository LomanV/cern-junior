GUIDE D'INSTALLATION et COMPILATION de SDL2/OPENGL
pour machines Apple.

1. Si tu ne l'as pas déjà, installe homebrew.
2. Installe avec homebrew SDL2:
	$brew install sdl2
3. Installe ensuite les librairies OpenGL:
	$brew install glfw
	$brew install glew
4. Telecharge TEST sur le git:
	$git clone https://github.com/raphael-vock/cern-junior/tree/master/TEST
5. Compile:
	$cd TEST
	$make
6. Si tout s'est bien passé, quand tu executes a.out tu auras une fenêtre avec des couleurs clignotantes.

REMARQUES:
- Les headers OpenGL/SDL sont différents sur les machines Apple que sur Windows/Linux. Il faut donc inclure les headers suivants:
	#include <SDL2/SDL.h>
	#include <SDL2/SDL_opengl.h>
	#include <OpenGL/gl.h>	

- La compilation de code source utilisant SDL2 nécessite qu'on indique au linker où chercher la librairie SDL2, donc on utilise le flag -lSDL2 dans g++.

- OpenGL quant à lui est un "framework" donc on utilise le flag -framework OpenGL
